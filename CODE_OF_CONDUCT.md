# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) verbatim. The full text, including the standards, scope, enforcement guidelines, and attribution, is hosted at the canonical URL above.

## Summary

The maintainers and contributors of `compose-performance-skills` are committed to making participation in this project a welcoming and productive experience for everyone. The Contributor Covenant defines the standards that apply across all project spaces: issues, pull requests, discussions, social media interactions tagged with this project, and any in person or online events the project organizes.

## Reporting

If you observe behavior that violates the Contributor Covenant, report it privately to the project maintainer at **skydoves2@gmail.com**. Reports stay confidential. The maintainer follows the Enforcement Guidelines in section 4 of the Contributor Covenant when deciding on a response.
